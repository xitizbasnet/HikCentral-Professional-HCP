# 1. Introduction

The HikCentral Professional Control Client is a desktop-based Video Management Software (VMS) used to monitor and manage CCTV camera footage within the organization.

Through this application, authorized personnel can:

* Access live camera views
* Monitor real-time activities
* View recorded footage (if permitted)
* Manage surveillance operations from multiple locations

This document outlines the step-by-step process to view live CCTV footage using the HikCentral Professional Control Client.

---

# 2. Purpose

The purpose of this document is to guide IT staff and authorized users through the correct procedure to:

* Launch the HikCentral Control Client application
* Access live CCTV camera feeds
* Search and select specific cameras for monitoring
* Properly close the application after use

This ensures:

* Consistent operation
* System security
* Proper handling of surveillance systems
* Controlled access to CCTV monitoring

---

# 3. Prerequisites

Before proceeding, ensure the following:

* HikCentral Professional Control Client is installed on your system
* You have valid administrator or user login credentials
* Network connectivity to the HikCentral Server is active

---

# 4. Steps to View Live CCTV Footage

---

# 4.1 Launch the Application

## Step 1: Open Start Menu

* Go to the **Start Menu** on Windows

---

## Step 2: Search Application

* Search for:

  * **HikCentral Professional Control Client**

---

## Step 3: Run as Administrator

* Right-click the application
* Select:

```text id="yq9v2m"
Run as Administrator
```

---

## Step 4: Enter Credentials

### Table 1 – User Credentials

| Field    | Value                 |
| -------- | --------------------- |
| Username | `Admin`               |
| Password | `*******************` |

---

## Step 5: Login

* Click **Login**
* Wait for system authentication to complete

---

# 4.2 Dashboard Overview

After successful login:

The **Dashboard Interface** will appear, providing access to:

* Live camera feeds
* Recorded video (if permitted)
* System alerts
* Monitoring tools
* Surveillance management features

---

# 4.3 Access the Monitoring Section

## Step 1: Open Monitoring

From the left navigation menu:

* Click **Monitoring**

---

## Step 2: Search Camera

Use the search bar to find cameras by:

* Name
* Location
* Department

### Example

```text id="c3n7kp"
Office_Center
```

---

## Step 3: Select Camera

* The matching result will appear below the search bar

---

# 4.4 View the Live Feed

## Step 1: Open Live View

* Double-click the camera result (e.g., `Office_Center`)

---

## Step 2: Monitor Feed

The **Live View** will appear in the dashboard.

Users can:

* View real-time footage
* Maximize screen view
* Adjust layout
* Switch camera views

---

# 4.5 Step to Close the Application

When monitoring is complete:

## Step 1: Close Application

* Click the **Close (X)** button (top-right corner)

---

## Step 2: Confirmation

* A confirmation dialog will appear

---

## Step 3: Confirm Exit

* Click **OK** to safely exit the system

---

> ⚠️ **Important Note:** Always close sessions properly to prevent unauthorized access and ensure system security.
