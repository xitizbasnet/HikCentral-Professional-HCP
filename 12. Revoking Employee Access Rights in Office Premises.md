# 1. 🧾 Introduction

This document defines the standardized process for managing and restricting physical access for employees whose service period has ended. It ensures that user access via fingerprint and card authentication is automatically disabled after the defined effective period.

Once this procedure is completed, the user will retain access to office resources only until **23:59:00 of the current day**. After this time, all biometric and card-based access within the office premises will be fully disabled.

---

# 2. 🎯 Purpose

The purpose of this SOP is to:

* 🔐 Maintain physical security within office premises
* ⏱ Ensure timely revocation of access for resigned employees
* 📋 Standardize the access termination process for the IT Department

---

This procedure applies to:

* 👨‍💻 IT Department

* 👥 All employees who have resigned or completed their tenure

# 6. ⚙️ Standard Operating Procedure (SOP)

---

## 6.1 🔑 Login to HCP System

1. Open a web browser 🌐
2. Enter the system URL: [https://172.16.1.1/#/](https://172.16.1.1/#/)
3. Log in using administrator credentials

   * Username: admin
   * Password: **********
4. Click **Login** to access the dashboard

---

## 6.2 🔍 Search for User

1. From the dashboard, click on **Person**
2. User details dashboard will be displayed
3. Use the **filter option** to locate the user
4. An empty search field will appear to enter user details
5. In the filter popup, enter First Name: **Resigned**
6. Click **Filter** 🔎
7. Search results will be displayed as illustrated

📌 *Note: Ensure correct filter input to avoid incorrect user selection.*

---

## 6.3 👁 View User Details

1. From search results, click on **User ID**
2. The user dashboard will open with full details

---

## 6.4 📅 Modify Effective Period

1. Locate the **Effective Period** section
2. Click on the 📆 calendar icon
3. Change the **end date to today’s date**
4. Click **OK** to confirm

✔ This ensures access expires at the end of the working day.

---

## 6.5 💾 Save Configuration

1. Click on the **Save (💾)** icon
2. System will redirect back to dashboard

✔ This confirms the configuration has been successfully saved.

---

## 6.6 🚪 Logout Procedure

1. Click on the **Username** in the top menu
2. A popup menu will appear:

   * 🔑 Change Password
   * 🚪 Logout
3. Select **Logout**
4. Click **OK** to confirm

---

# 7. 📊 Expected Outcome

* ⏳ User access remains active until **23:59:00 of the current day**
* 🧠 Fingerprint access will be disabled after expiry
* 💳 Card access will be disabled after expiry
* 🚷 User will not be able to access office premises
