# 1. Introduction

As part of the daily monitoring process, it is essential to check the HDD status on every NVR (Network Video Recorder) each morning. This task helps assess storage health and ensures the continued proper functioning of the surveillance system.

The HDD status can be checked through the HikCentral Professional Client (HCP) web interface using the following portal:

```text id="fjlwm8"
https://192.168.1.1/#/
```

---

# 1.1 Abstract

The objective of this document is to provide a step-by-step guide for checking HDD status using the HikCentral Professional Client (HCP) web interface.

By following the instructions outlined in this guide, IT personnel will be able to:

* Monitor HDD health efficiently
* Detect storage-related issues early
* Maintain system reliability
* Prevent NVR storage failures

---

# 2. Procedure

# 2.1 Login to the HCP Web Interface

## Step 1: Open the Web Portal

Open **Firefox** (or any preferred web browser) and navigate to the following URL:

```text id="4hmdhg"
https://192.168.1.1/#/
```

---

## Step 2: Enter Login Credentials

### Table 1 – Login Credentials

| Field    | Value                 |
| -------- | --------------------- |
| Username | `Admin`               |
| Password | `*******************` |

> ⚠️ **Security Notice:** Ensure login credentials are securely managed and accessible only to authorized personnel.

---

## Step 3: Login to the System

After entering the credentials:

* Click the **Login** button to proceed.

---

# 2.2 Navigate to the Maintenance Section

## Step 4: Open Maintenance Menu

After successful login:

* Locate the top navigation menu bar.
* Click on the **Maintenance** tab.

---

# 2.3 Access the Resource Status

## Step 5: Open Resource Status

From the left-side navigation menu:

* Select **Resource Status**

---

## Step 6: Open Encoding Devices

Under **Resource Status**:

* Click on **Encoding Devices**

This will open the dashboard containing HDD and device health information.

---

# 2.4 View HDD Status

## Step 7: Check Disk Status

On the dashboard page:

* Locate the **Disk Status** section.

---

## HDD Status Verification

| Status                       | Meaning                                                    |
| ---------------------------- | ---------------------------------------------------------- |
| Normal                       | HDD is functioning properly                                |
| Any Status Other Than Normal | Further investigation or corrective action may be required |

---

## Recommended Action

### If Status is `Normal`

* No immediate action is required.

### If Status Shows `Exception` or Any Other Warning

* Inform the IT Team immediately.
* Coordinate with the vendor for further troubleshooting and resolution.

> ⚠️ **Important:** Early detection of HDD issues helps prevent storage failure and recording interruptions.

---

# 2.5 Logging Out

## Step 8: Open Logout Menu

After checking the HDD status:

* Click on the login/profile icon located at the top-right corner of the page.

---

## Step 9: Logout from the System

* Select the **Logout** option.

---

## Step 10: Confirm Logout

* Click **OK** to complete the logout process.

---

# 3. Conclusion

By following this procedure, HDD status can be monitored daily as part of routine NVR system maintenance.

This process helps:

* Detect storage-related problems early
* Maintain system health and stability
* Prevent unexpected HDD failures
* Ensure uninterrupted surveillance recording operations

Regular monitoring and timely escalation of abnormal HDD conditions are critical for maintaining a reliable surveillance infrastructure.
