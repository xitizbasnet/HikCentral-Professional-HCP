# 📘 Security Monitoring System - Smart Wall Camera Configuration via HikCentral Professional Control Client (HCP)  

## 1. 🎯 Purpose
The purpose of this document is to define the standardized operational procedure for configuring camera display feeds on the security Smart Wall using the HikCentral Professional (HCP) Control Client.  

It provides detailed guidance for:
- Adding camera feeds  
- Removing camera feeds  
- Activating camera feeds on **Monitor 1**  

Monitor 1 is configured for **automated sequential display (auto-switching)** of multiple surveillance cameras.  

✅ This ensures:
- Consistent monitoring across high-priority surveillance zones  
- Operational continuity in the Security Operations Room  

---

## 2. 📌 Scope
This procedure applies to:
- 👮 Security personnel operating the Surveillance Control Room  
- 💻 IT Department staff responsible for system configuration and maintenance  
- 🔐 Authorized personnel overseeing surveillance infrastructure  

📍 **Scope Limitation:**  
This document is currently limited to **Monitor 1 configuration**.  
Other monitors retain fixed camera feeds and are documented separately.  

---

## 3. 🖥️ System Description

The Security Management Room utilizes an integrated video monitoring solution comprising **four active display monitors** mounted in a **2×2 layout**.

### 📊 Monitor Mapping

| Physical Monitor | Position     | Smart Wall Identifier | Functionality                                  |
|-----------------|-------------|-----------------------|-----------------------------------------------|
| Monitor 1       | Upper Left  | 100                   | Auto-switching view of multiple cameras       |
| Monitor 2       | Upper Right | 200                   | Fixed surveillance view                      |
| Monitor 3       | Lower Left  | 300                   | Fixed surveillance view                      |
| Monitor 4       | Lower Right | 400                   | Fixed surveillance view                      |

🔄 **Monitor 1 Behavior:**  
- Rotates camera feeds approximately every **15 seconds**  
- Provides continuous visual sweep of surveillance zones  

⚠️ **Important Note:**  
Smart Wall configuration must always be done in **Private View Mode** to prevent accidental changes to the live operational view.

## 4. 🔐 Access Procedure – HikCentral Control Client Login

Follow these steps to access the system:

a) Launch the **HikCentral Professional Control Client** from Windows search  
b) The login window will appear  
c) Enter the following credentials:  

```text
Username: admin
Password: ********
````

d) Click **Login** to proceed

e) Upon successful authentication, the main dashboard interface will load

f) Navigate to **Smart Wall Control** for centralized monitor and camera management

---

## 5. ⚙️ Navigating to Smart Wall View Configuration

a) Within the Smart Wall Control interface, locate **Smart Wall 1** and select it

b) Upon successful selection, the main dashboard interface will load

c) Click on **“View” mode**

d) Under View Mode settings, ensure **Private View** is selected:

* 🔧 **Private View** — used for editing, staging, and saving configurations
* 👁️ **Public View** — used for real-time display only

e) Proceed only after confirming **Private View mode is active**

⚠️ **Important:**
Modifications performed under **Public View** will not persist and may disrupt live monitoring.

f) The Security Management Room utilizes an integrated video monitoring solution comprising four active display monitors mounted in a 2x2 layout. Each monitor is uniquely mapped to a Smart Wall reference identifier for operational clarity:

| Physical Monitor | Position    | Smart Wall Identifier | Functionality                           |
| ---------------- | ----------- | --------------------- | --------------------------------------- |
| Monitor 1        | Upper Left  | 100                   | Auto-switching view of multiple cameras |
| Monitor 2        | Upper Right | 200                   | Fixed surveillance view                 |
| Monitor 3        | Lower Left  | 300                   | Fixed surveillance view                 |
| Monitor 4        | Lower Right | 400                   | Fixed surveillance view                 |

## 6. 🧭 Monitor Identification and Selection

To begin configuration:

a. In the Smart Wall diagram, click **Monitor 1 (100)**

b. Selection confirmation:

* 🟢 A **green border highlight** appears around the monitor

c. Once activated:

* Only **Monitor 1** will be affected
* Other monitors remain unchanged

⚠️ **Important:**
This prevents unintended overwrite of adjacent monitor configurations.

---

## 7. ➖ Removing a Camera Feed from Monitor 1

This procedure is used when:

* A camera is no longer relevant
* Temporary deactivation is required

Steps:

d. Double-click any tile on Monitor 1

e. Click **View Details**

f. The **Multi-Window Auto-Switch Configuration** screen opens

g. From the resource list:

* Identify and select the camera
* Example:

  ```text
  Office_Decoder_2F_LADDER_01
  ```

h. Click the **Delete icon** above the resource list

i. Confirm removal from the list

j. Click **OK**

k. Save configuration **twice**:

* 💾 First: Save icon on monitor layout
* 💾 Second: Select **Save** (not Save As)

✅ **Outcome:**
The camera will no longer appear in the auto-switch sequence.

---

## 8. ➕ Adding a Camera Feed to Monitor 1

Used for:

* Re-adding removed cameras
* Adding new surveillance units

Steps:

i. Select **Monitor 1 → View Details**

ii. Open **Multi-Window Auto-Switch Configuration**

iii. Click the **➕ Add icon**

iv. Choose signal source:

* Device signal source
* Network signal source

📌 **Note:** Select **Network Signal Source** to view available cameras

v. Use the search option

vi. Enter camera identifier:

```text
Example: Office_Decoder_2F_LADDER_01
```

vii. Select from results

viii. Click **OK**

ix. Perform double save

x. Save configuration **twice**:

* 💾 First: Save icon
* 💾 Second: Save (not Save As)

✅ **Outcome:**
Camera is added to the rotation cycle.

---

## 9. ▶️ Activating Live Camera Preview

After configuration, preview may be inactive by default.

Steps:

a. Return to **Smart Wall Dashboard**

b. Locate **Display Camera Image on Client** (bottom-right)

c. Toggle status:

* ⚪ Grey = Disabled
* 🟢 Green = Enabled

d. Once enabled:

* Monitor 1 begins live display

📌 **Note:**
This must remain enabled for verification of camera activity.

## 10. 🚪 System Exit Procedure

a. Click **Close (X)** at top-right

b. Confirm shutdown by clicking **OK**

c. Ensure application is fully closed

Always leave the terminal in a **secure state** after use.
