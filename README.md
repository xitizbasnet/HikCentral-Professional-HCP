## 📌 About HikCentral Professional

HikCentral Professional (HCP) is a centralized security management platform developed by Hikvision. It is used for:

- Video Surveillance Management
- Access Control Systems
- Attendance Management
- Alarm Monitoring
- Device Management & Integration
