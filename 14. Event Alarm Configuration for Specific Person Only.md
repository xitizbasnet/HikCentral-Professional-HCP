# 📌 Overview

## Event/Alarm Configuration for Specific Person Only

**Date:** Wednesday, March 5, 2025

### 🎯 Objective

The purpose of this document is to outline the process for configuring events and alarms within the Hik-Central Professional (HCP) system, including:

* Event and alarm email template creation
* Email notification configuration
* User-specific notification delivery
* Access control event communication

This process enables targeted notifications for specific personnel and ensures efficient monitoring and communication within the infrastructure and security environment.

# 2. Access the System

## Step 1: Access the System

After successful login:

* Navigate to the **System Settings** using the upper-side navigation menu.

---

## Step 2: Access the Email Configuration

### Navigate to the Email Section

From the left-side navigation menu:

* Locate and click on the **Email** option to access email-related settings.

---

## Select Event and Alarm Email Template

From the dropdown menu under **Email**:

* Select **Event and Alarm Email Template**

---

## Step 3: Add a New Email Template

### Create a New Template

* Click the **Add** button to create a new event and alarm email template.

---

### Enter Template Name

In the **Template Name** field:

* Provide a descriptive template name.

### Example

```text id="mrl4tz"
HR Notification
```

---

## Step 4: Add User Emails

### Enter User Email Addresses

In the **User Email** field:

* Enter the email addresses of users who should receive notifications.

### Notes

* Up to **64 email addresses** can be added.
* Ensure all email addresses are valid and correctly formatted.

---

## Step 5: Define the Subject Line

### Configure Email Subject

In the **Subject** field:

* Specify the email notification subject.

This subject can represent:

* General notifications
* Event triggers
* Alarm alerts
* Access control events

---

## Step 6: Save the Template

After completing the required fields:

* Template Name
* User Emails
* Subject

Click:

* **Save**

This will successfully save the new email template configuration.

---

# 3. Test the Email Template

## 3.1 Test the Template

After saving the template:

* Perform a test to ensure the email template functions correctly.

### Verification Checklist

* Email delivery is successful
* Subject formatting is correct
* Intended recipients receive notifications
* Template content appears as expected
