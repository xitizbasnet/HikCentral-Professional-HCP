# 2. Access the Event and Alarm Configuration Section

## 2.1 Navigate to Event and Alarm

From the main dashboard:

* Go to the **Event and Alarm** section from the top menu.

---

## 2.2 Open Event and Alarm Configuration

From the left-hand navigation menu:

* Select **Event and Alarm Configuration**

---

## 2.3 Open Normal Event and Alarm

* Click on **Normal Event and Alarm** from the left navigation panel.

---

## 2.4 Add New Event Configuration

* Press the **Add** button to initiate a new event and alarm configuration setup.

---

# 3. Configure Event and Alarm Settings

## 3.1 Event and Alarm Dashboard

After clicking **Add**, the system will redirect to the:

* **Add Event and Alarm Configuration Dashboard**

Proceed with the required setup.

---

## 3.2 Access Control Configuration

### Steps

* Select the **Access Control** section.
* Under the Access Control section, select **Door**.

---

## 3.3 Select Event Type

Navigate to the submenu and:

* Select **Normal Card Swiping**

---

## 3.4 Select Authentication Methods

Choose the required authentication methods.

### Selected Methods

* Fingerprint
* Card

> 📌 For the current configuration, both authentication methods are enabled for vault room access.

---

## 3.5 Location Selection

From the **Location** dropdown:

* Select **Office Room – 1st Floor**
* Check the confirmation box to apply the selection.

---

# 4. Configure Email Notification Settings

## 4.1 Modify Subject Format

Under **Email Settings**, verify that the subject format is configured as:

```text
{$Source} - {$Area} - {$Triggering Event}
```

### Required Modification

Remove:

```text
{$Area} - {$Triggering Event}
```

### Final Subject Format

```text
{$Source}
```

---

## 4.2 Description Configuration

* Provide a clear and meaningful description for the configured event.

---

## 4.3 Enable Ignore Repetitive Events

Enable the following option:

* **Ignore Repetitive Events/Alarms**

### Verification

* Ensure the activation icon turns **green** after enabling.

---

## 4.4 Configure Email Action

### Steps

* Select **Add Linkage Action**
* Choose **Send Email**

This ensures email notifications are triggered automatically during event activation.

---

## 4.5 Email Schedule Configuration

From the dropdown menu:

* Select **All-Day-Template**

### Schedule Configuration

Choose:

```text
Vault Time (after 6 PM before 9 AM)
```

---

## 4.6 Alarm Trigger Setup

Enable the following option:

* **Trigger Alarm**

### Verification

* Ensure the activation icon changes to **green** after enabling.

---

## 4.7 Email Recipient Configuration

Add the following recipient:

* **Shilipi Saxena**

### Finalize Recipient

* Check the confirmation box to apply the recipient configuration.

---

## 4.8 Finalize Configuration

* Press the **Add** button to complete the event and alarm setup successfully.

---
