# 📖 Overview

This guide provides step-by-step instructions for changing your password in the HCP (Hikvision Central Professional) Server.

Following this procedure helps ensure:

* Secure access to your account
* Compliance with password security policies
* Successful password updates without account interruption

---

# ⚠️ Security Notice

> 🔒 Password Security Best Practices

When creating a new password:

* Use a strong and unique password
* Avoid reusing previous passwords
* Do not share your password with others
* Follow organizational password complexity requirements

# 🚀 Procedure: Change Password in HCP Server

---

# Step 1 — Log In to the HCP Server

1. Open your preferred web browser.
2. Navigate to the HCP login page:

```text id="d8v3mt"
https://172.16.1.1/#/
```

3. Enter your:

   * Username
   * Current password

4. Click:

```text id="y0f2qe"
Login
```

to access the system.

---

# 🔔 Step 2 — Change Password (If Prompted)

After logging in, you may see a pop-up notification requesting a password change.

---

## If the Password Change Pop-Up Appears

### Step 2.1 — Open Password Change Window

1. Click:

```text id="o6k3pz"
Change Password
```

within the pop-up window.

---

## Step 2.2 — Enter Current Password

1. In the:

```text id="z4j7lb"
Old Password
```

field, enter your current password.

---

## Step 2.3 — Enter New Password

1. In the:

```text id="a9m2xy"
New Password
```

field, enter your desired new password.

> 💡 Tip
> Ensure the new password meets all required complexity standards, such as:
>
> * Minimum character length
> * Uppercase and lowercase letters
> * Numbers
> * Special characters

---

## Step 2.4 — Confirm New Password

1. Re-enter the new password in the:

```text id="k1v8dn"
Confirm New Password
```

field.

This confirms and verifies the password entry.

---

# 🛠️ Step 3 — Manual Access to Change Password

If no pop-up notification appears, you can manually change your password using the settings menu.

---

## Open Password Settings

1. Navigate to the account/settings section.
2. Locate and select:

```text id="g3w6tr"
Change Password
```

---

# 🔑 Step 4 — Enter Old and New Password

Complete the password update form using the following steps.

---

## Enter Current Password

1. In the:

```text id="v8t4ph"
Old Password
```

field, enter your current password.

---

## Enter New Password

1. In the:

```text id="u2m7kc"
New Password
```

field, enter your desired new password.

> 💡 Tip
> Ensure the password follows your organization’s password policy requirements.

---

## Confirm the New Password

1. Re-enter the new password in the:

```text id="s5j9wx"
Confirm New Password
```

field.

---

# 💾 Step 5 — Submit the Changes

After completing all required fields:

1. Click:

```text id="x7d1me"
Submit
```

or

```text id="q0v5rf"
Save
```

to apply the password changes.

---

# ✅ Step 6 — Re-Login to Confirm the Change

To verify the password update was successful:

1. Log out of the HCP server.
2. Return to the login page.
3. Log in again using:

   * Your username
   * Your newly updated password

> ✅ Successful login confirms the password was updated correctly.

---

# 🔍 Step 7 — Troubleshooting

If you experience issues during or after the password change process, review the following troubleshooting recommendations.

---

## Unable to Log In After Password Change

Verify the following:

* The new password was entered correctly
* Password confirmation matched exactly
* The new password meets all complexity requirements

---

## Password Change Not Accepted

Possible causes:

| Issue             | Description                            |
| ----------------- | -------------------------------------- |
| Weak password     | Password does not meet security policy |
| Password mismatch | Confirmation password differs          |
| Session timeout   | Login session expired during update    |
| Browser issue     | Cache or browser compatibility issue   |

---

## Recommended Actions

* Retry the password update process
* Clear browser cache if needed
* Ensure Caps Lock is disabled
* Use a supported web browser

---

## Contact IT Support

If the issue persists:

> 📞 Contact the IT Department for further assistance.
