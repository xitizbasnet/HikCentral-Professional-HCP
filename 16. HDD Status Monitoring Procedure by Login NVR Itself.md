# 1. Introduction

As part of the daily monitoring process, it is essential to check the HDD status on every NVR (Network Video Recorder) each morning. This task helps assess storage health and ensures the continued proper functioning of the surveillance system.

The HDD status can be checked through the HikCentral Professional Client (HCP) web interface using the following portal:

```text id="p5vfwp"
http://192.168.1.250/doc/index.html#/portal/login
```

# 1.1 Abstract

The objective of this document is to provide a step-by-step guide for checking HDD status using the HikCentral Professional Client (HCP) web interface.

By following the instructions outlined in this guide, IT personnel will be able to:

* Monitor HDD health efficiently
* Identify storage-related issues early
* Maintain system reliability
* Ensure uninterrupted surveillance operations

# 2. Procedure

# 2.1 Login to the HCP Web Interface

## Step 1: Open the Web Portal

Open **Firefox** (or any preferred web browser) and navigate to the following URL:

```text id="6o6g7l"
http://192.168.1.250/doc/index.html#/portal/login
```

---

## Step 2: Enter Login Credentials

### Table 1 – Login Credentials

| Field    | Value      |
| -------- | ---------- |
| Username | `Admin`    |
| Password | `********` |

> ⚠️ **Security Notice:** Ensure login credentials are handled securely and accessible only to authorized personnel.

---

## Step 3: Login to the System

After entering the credentials:

* Click the **Login** button to proceed.

---

# 2.2 Navigating to Storage Management

## Step 1: Access the NVR Dashboard

After successful login:

* The **NVR Dashboard** will be displayed.

---

## Step 2: Open Configuration

* Click on **Configuration**

---

## Step 3: Navigate to Storage

* Navigate to the **Storage** section.

---

## Step 4: Open Storage Management

Under the **Storage** section:

* Select the dropdown option
* Choose **Storage Management**

---

## Step 5: Review HDD Dashboard

The **Storage Management Dashboard** will appear and display the HDDs currently used in the NVR.

### Current Configuration

* Total Storage Units: **5**
* Capacity per HDD: **8 TB**

> 📌 Verify that all configured storage units are visible and functioning properly.

---

# 2.3 Checking HDD Status

## Step 1: Verify Synchronization State

In the dashboard, locate the:

* **Synchronization State**

### Status Conditions

| Status                     | Description                                |
| -------------------------- | ------------------------------------------ |
| Normal                     | System is functioning correctly            |
| Exception / Other Messages | Consult the IT Team and Vendor immediately |

---

## Step 2: Verify HDD Status

Locate the:

* **Status** field

### Status Verification

| Status                     | Action Required                |
| -------------------------- | ------------------------------ |
| Normal                     | No action required             |
| Exception / Other Messages | Escalate to IT Team and Vendor |

> ⚠️ **Important:** Any abnormal HDD status should be reported immediately to avoid storage failure or recording interruptions.

---

# 2.4 Logging Out from the Web Portal

## Step 1: Logout from the Portal

* Click on **Logout**

---

## Step 2: Confirm Logout

* Click **OK** to confirm logout.

---

# 3. Conclusion

By following this procedure, the HDD status can be monitored daily as part of routine system maintenance.

This process helps:

* Detect storage-related issues early
* Maintain NVR system health
* Prevent unexpected HDD failures
* Ensure uninterrupted surveillance recording operations

Regular monitoring and timely escalation of abnormal HDD conditions are critical for maintaining a reliable and secure surveillance infrastructure.
