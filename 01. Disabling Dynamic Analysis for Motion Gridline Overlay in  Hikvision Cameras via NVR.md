# 📘 Disabling Dynamic Analysis for Motion (Gridline Overlay) in Hikvision Cameras via NVR

## 1. 📌 Introduction

This document provides a **step-by-step procedure** for disabling the **Dynamic Analysis for Motion (Gridline Overlay)** feature on Hikvision cameras.

> ⚠️ **Important Notes**
- This configuration **cannot be modified** from the **HikCentral Professional (HCP) Server**.
- To disable this feature, you must log in directly to the **Network Video Recorder (NVR)**.
- In this example, the configuration is performed on **NVR 02**.

---

## 2. 🌐 Accessing the NVR

Follow the steps below to access the NVR interface:

1. Open a web browser and navigate to the following URL:
```

http://192.168.1.25/doc/index.html#/portal/login

```

2. Enter the login credentials:
- **Username:** `admin`  
- **Password:** `*****`

3. Click **Login** to access the NVR interface.

> 🔐 **Security Note:** Ensure credentials are handled securely and not shared with unauthorized personnel.

---

## 3. ⚙️ Navigating to the Configuration Menu

After successfully logging in:

1. The system dashboard will be displayed.
2. From the **top menu**, select **Configuration**.
3. In the **left-hand sidebar**, click **Event**.
4. Under the **Event** section, select **Basic Event**.

---

## 4. 🛠️ Disabling Dynamic Analysis for Motion

Follow these steps to disable the feature:

1. In the **Camera Selection** dropdown, choose the appropriate camera.  
- Example: **D1 Camera** (from a total of D1–D64 cameras)

2. Locate the option: **Enable Dynamic Analysis for Motion**  
- The radio button will appear **🟢 Green** if the feature is currently enabled.

3. Click the toggle to disable the feature.  
- Once disabled, the icon will turn **⚪ Grey**.

4. Click **Save** to apply and store the updated configuration.

> ℹ️ **Note:**  
> After disabling the feature, the gridline overlay will **no longer appear in the live view**, although it **may still appear in the camera preview**.  
> This is **normal behavior**.

---

## 5. 🔄 Logging Out and Verification

To ensure changes are properly saved:

1. Click **Logout** located in the **top-right corner** of the navigation bar.
2. Confirm the logout action by selecting **OK**.

> ✅ **Verification Tip:**  
> Re-login and check the camera live view to confirm that the gridline overlay is no longer displayed.

---

## 6. ✅ Conclusion

You have successfully disabled the **Dynamic Analysis for Motion (Gridline Overlay)** feature on the selected Hikvision camera through the NVR interface.
