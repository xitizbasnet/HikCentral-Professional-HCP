# 📖 Overview

This document explains how to retrieve authentication log records for a specific person within the HCP Access Control System.

The guide covers:

* Accessing the HCP portal
* Searching authentication records
* Filtering logs by specific users
* Applying custom date and time filters
* Exporting log results to Excel format

# 🚀 Procedure: Retrieve Authentication Logs

---

# Step 1 — Login to HCP

1. Open a web browser.
2. Navigate to:

```text id="m8qz4z"
https://172.16.1.1/#/
```

3. Enter the administrator credentials.
4. Click **Login**.

---

# Step 2 — Open Access Control Module

After successful login:

1. Navigate to:

```text id="s0a1l5"
Access Control
```

---

# Step 3 — Open Person Authentication Record

1. In the left-side navigation/search panel:
2. Select:

```text id="jlwm9x"
Person Authentication Record
```

---

# Step 4 — Select Person Search Method

Under the search section:

1. Locate:

```text id="dhhjzx"
Person/Visitor
```

2. Select:

```text id="r0e2bn"
Select Person
```

under the **Search Method** section.

---

# Step 5 — Search for a Specific User

1. Click:

```text id="lz0e6z"
Select Person
```

2. Search for the required user by name.

### Example

```text id="x5s8wz"
Xitiz Basnet
```

3. Select the user from the results list.
4. Click:

```text id="qrfqmr"
Add
```

5. Click:

```text id="zjlwm8"
Search
```

---

# 📊 View Search Results

The authentication records for the selected user will appear in the results panel.

---

# 🕒 Filter Logs by Specific Date and Time

To retrieve logs for a specific time frame:

---

## Open Time Filter Menu

1. Navigate to the top menu section.
2. Locate the:

```text id="2aodk7"
Time
```

menu.

---

## Select Custom Time Interval

1. Open the time filter dropdown.
2. By default, it may display:

```text id="u4m3ln"
Today
```

3. Select:

```text id="8pl6xw"
Custom Time Interval
```

---

## Configure Date & Time Range

1. Choose:

   * Start Date & Time
   * End Date & Time

### Example

```text id="89q7qx"
October 29, 2024
```

2. Click:

```text id="x0n13y"
OK
```

3. Click:

```text id="tjlwm9"
Search
```

---

# 📈 View Filtered Results

The filtered authentication log results will appear on the right-side panel.

---

# 📤 Export Search Results to Excel

After the search results are displayed:

1. Navigate to the top menu bar.
2. Select:

```text id="o2p6t8"
Export
```

3. Choose:

```text id="3r7kph"
Excel
```

4. Enter your user credentials if prompted.
5. Save the export file.

---

# ✅ Export Confirmation

After the export completes successfully, a confirmation popup message will appear indicating:

```text id="8w9zef"
Operation completed
```

# 🔍 Troubleshooting Tips

## Unable to Find User

* Verify the user exists in HCP
* Check spelling of the username
* Confirm user synchronization status

---

## Export Option Not Working

Possible causes:

* Insufficient permissions
* Browser popup blocking
* Export service interruption

---

## No Search Results Found

Verify:

* Correct date/time range selected
* User has authentication records
* Filters are not overly restrictive

---

# 🔒 Security Recommendations

* Do not share administrator credentials publicly
* Restrict access to exported log files
* Store reports in secured directories
