# 📖 1. Introduction

This document defines the standard operating procedure for retrieving ANPR (Automatic Number Plate Recognition) vehicle number plate records using the Hik-Central Professional (HCP) system.

The purpose of this guide is to assist the IT team in:

* Accessing ANPR dashboards
* Searching vehicle records
* Filtering camera-based logs
* Exporting reports for analysis and auditing

---

# 🔐 2. System Login Procedure

---

## 🌐 2.1 Access URL

Open a web browser and navigate to:

```text id="hcp_url_anpr"
https://172.16.1.1/#/
```

---

## 🔑 2.2 Enter Credentials

Use the following login details:

| Field    | Value        |
| -------- | ------------ |
| Username | admin        |
| Password | ************ |

---

## 🚪 2.3 Login

1. Enter credentials in the login form.
2. Click:

```text id="hcp_login_anpr"
Login
```

3. After successful authentication, the **HCP dashboard** will be displayed.

---

# 🚗 3. Navigation to ANPR Records

Follow the steps below to access vehicle number plate data:

---

## Step 1 — Open ANPR Module

Select:

```text id="anpr_menu_01"
ANPR
```

from the top navigation bar.

---

## Step 2 — Open Vehicle Search

From the left panel, select:

```text id="vehicle_search_01"
Vehicle Passing Search
```

---

## Step 3 — Select Camera Source

Choose the required ANPR camera, for example:

```text id="camera_example_01"
ANPR_EXIT
```

---

## Step 4 — Set Filters

Define search parameters:

* Date range
* Time range
* Camera selection

---

## Step 5 — Execute Search

Click:

```text id="search_btn_01"
Search
```

to retrieve results.

---

# 📊 4. Viewing and Exporting Search Results

---

## 📌 Viewing Results

After executing the search:

* Results will appear on the **right-hand side panel**
* Each entry includes:

  * Vehicle number plate
  * Timestamp
  * Camera location
  * Snapshot (if available)

---

## 📤 Exporting ANPR Data

To export results:

1. Click **Export** from the top results menu.
2. Select export format:

```text id="export_format_01"
Excel
```

3. Enter administrator password when prompted.
4. Click **Save**.

---

## 📁 Export Output Location

The exported file will be saved in:

```text id="download_path_01"
Downloads Folder
```

---

## ✅ Verification

After download:

* Open the file
* Confirm vehicle records are correctly exported
* Validate date/time accuracy

---


2. Select:

```text id="logout_01"
Logout
```

3. Confirm by clicking:

```text id="confirm_logout_01"
OK
```

# 📌 6. Conclusion

Following this procedure ensures that the IT team can:

* Access ANPR data securely
* Perform vehicle number plate searches efficiently
* Export reports for analysis and reporting
* Maintain audit-ready records from HCP system

This process ensures accurate and reliable retrieval of vehicle movement data for operational and security purposes.
