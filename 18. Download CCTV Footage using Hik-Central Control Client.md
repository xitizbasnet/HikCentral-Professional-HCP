# 1. Introduction

The HikCentral Professional Control Client is a desktop-based Video Management Software (VMS) used to monitor and manage CCTV camera footage within the organization.

Through this application, authorized personnel can:

* Access live camera feeds
* Monitor real-time activities
* Retrieve historical recordings
* Export surveillance footage
* Manage CCTV surveillance operations

This document outlines the step-by-step process for viewing live CCTV footage and exporting recorded video using the HikCentral Professional Control Client.

---

# 2. Purpose

The purpose of this document is to guide IT staff and authorized users through the correct procedures to:

* Launch the HikCentral Control Client application
* Access live CCTV camera feeds
* Search and select specific cameras
* View recorded footage
* Export CCTV footage in AVI format
* Properly close the application after use

Following these procedures ensures:

* Consistent operation
* System security
* Proper surveillance handling
* Secure evidence management

---

# 3. Prerequisites

Before proceeding, ensure the following requirements are met:

* HikCentral Professional Control Client is installed on the system
* Valid administrator or user login credentials are available
* Network connectivity to the HikCentral Server is active
* Appropriate permissions are assigned for playback and export operations

---

# 4. Steps to View Live CCTV Footage

# 4.1 Launch the Application

## Step 1: Open the Start Menu

* Click the **Start Menu** on Windows.

---

## Step 2: Search for the Application

* Search for **HikCentral Professional Control Client**.

---

## Step 3: Run as Administrator

* Right-click the application icon.
* Select:

```text id="fgmry0"
Run as Administrator
```

> ⚠️ Running the application as Administrator ensures proper access privileges.

---

## Step 4: Enter Login Credentials

When prompted, enter the following credentials:

### Table 1 – User Credentials

| Username | Password              |
| -------- | --------------------- |
| `Admin`  | `*******************` |

> ⚠️ **Security Notice:** Ensure credentials are securely managed and shared only with authorized personnel.

---

## Step 5: Login to the System

* Click the **Login** button.

---

## Step 6: Authentication Process

* Wait for the system to authenticate your credentials successfully.

---

# 4.2 Dashboard Overview

After successful login:

* The main **Dashboard Interface** will appear.

The dashboard provides access to:

* Live camera monitoring
* Playback and recording
* System alerts
* Device management
* Surveillance tools

---

# 4.3 Access the Monitoring Section

## Step 1: Open Monitoring

From the left-side navigation menu:

* Click on **Monitoring**

---

## Step 2: Search for a Camera

Use the **Search Bar** to locate cameras by:

* Camera Name
* Location
* Department
* Area

### Example

```text id="3c7vme"
Office_Center
```

---

## Step 3: Select Camera

* The corresponding camera result will appear below the search field.

---

# 4.4 View the Live Feed

## Step 1: Open Live View

* Double-click the camera result from the search list.

### Example

```text id="zm8w43"
Office_Center
```

---

## Step 2: Monitor Live Feed

The selected camera’s **Live View** will appear in the main dashboard section.

Users can:

* Monitor real-time footage
* Maximize the viewing window
* Adjust layout settings
* Switch between camera streams

> 📌 Ensure monitoring sessions are conducted only by authorized personnel.

---

# 5. Steps to Export Recorded Footage (Playback and Download)

# 5.1 Access the Playback Option

To retrieve historical footage:

* Select the **Playback** option from the Control Client interface.

This feature allows users to:

* View recorded video streams
* Search historical recordings
* Export surveillance footage

---

# 5.2 Initiate Export Process

## Step 1: Open Export Function

From the top-right navigation menu bar:

* Locate the **Export** icon
* Click the icon to initiate the export process

---

## Step 2: Export Dashboard

* A new export dashboard panel will appear on the right side of the screen.

---

# 5.3 Explore Export Options

Within the export panel:

* Scroll down to explore additional export settings and options.

The interface includes:

* Calendar selection tools
* Time range configuration
* File format options
* Storage path selection

---

# 5.4 Set Date and Time Range

## Step 1: Open Calendar Tool

* Click the **Calendar Icon**

---

## Step 2: Configure Time Range

A new calendar pop-up will appear containing:

* Start Date and Time
* End Date and Time

---

## Step 3: Select Export Duration

* Select the desired footage time range for export.

---

## Step 4: Confirm Selection

* Click **OK** to save the selected date and time range.

---

# 5.5 Choose File Type

## Step 1: Select File Type

Locate the:

* **File Type** option

---

## Step 2: Select Export Format

Choose:

```text id="ijuy18"
AVI
```

---

## Recommended Format Information

> 📌 The AVI file format allows playback directly through Windows Media Player on Windows 10 and newer systems.

This format is recommended for:

* Standard viewing
* Archiving
* Investigation purposes
* Evidence storage

---

# 5.6 Choose Storage Location

## Step 1: Select Destination Folder

Choose the folder where exported footage will be saved.

---

## Default Storage Path

* The system may automatically suggest a default storage location.

---

## Optional Custom Storage

Users may manually change the save location according to:

* Departmental storage guidelines
* Investigation requirements
* Archival procedures

---

# 5.7 Save and Download the Footage

## Step 1: Start Export

After selecting:

* File Type
* Storage Location
* Time Range

Click:

* **Save**

to begin the download process.

---

## Step 2: Monitor Download Progress

* Click the **Activity** icon to monitor export progress.

---

## Step 3: Verify Completion

Once completed successfully:

* The exported footage will appear under the **Completed** section.

---

# 5.8 Access the Downloaded Footage

## Step 1: Open Downloaded File

* Double-click the name of the completed download.

---

## Step 2: Open Storage Folder

* The system will automatically open the folder containing the exported video files.

---

# 5.9 Step to Close the Application

When monitoring or exporting operations are complete:

## Step 1: Close the Application

* Click the **Close (X)** button at the top-right corner of the Control Client window.

---

## Step 2: Confirmation Dialog

* A confirmation dialog box will appear requesting reconfirmation.

---

## Step 3: Confirm Exit

* Click **OK** to safely exit the application.

---

## Step 4: Application Exit

* The application will close successfully.

> ⚠️ **Important:** Always close active sessions properly to maintain system security and prevent unauthorized access.
