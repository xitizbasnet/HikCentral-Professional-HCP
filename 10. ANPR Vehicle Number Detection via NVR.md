# 1️⃣ Introduction

This document defines the standard operating procedure (SOP) for retrieving ANPR (Automatic Number Plate Recognition) reports using the Network Video Recorder (NVR).

This process is required because an ANPR license is not available in **HikCentral Professional**. Therefore, ANPR data must be accessed directly through the NVR interface.

The purpose of this document is to guide the IT (Infrastructure & Security) team in:

- Accessing the NVR system
- Searching ANPR camera data
- Retrieving vehicle number plate records
- Viewing captured events accurately

# 3️⃣ Navigation to ANPR Records

To access ANPR vehicle detection records, follow these steps:

1. From the main navigation menu, click on **File**.  
2. Select the required ANPR camera (example: `D16 ANPR_Entry`).  
3. In the **Event Type** field, select **Vehicle Detection**.  
4. In the **Country** field, select **ALL**.  
5. Define the required **Start Date** and **End Date** for the search period.  
6. Click **Search** to retrieve ANPR detection results.  

📌 **Result Display:**  
After executing the search, all matching records will appear on the right-hand side of the screen.

---

# 4️⃣ Viewing Search Results

- To view detailed vehicle information, click on the **Live View** option.  
- After selecting **View**, the system will display the captured ANPR data.

---

# 5️⃣ Closing Live View

- After reviewing the required vehicle details, close the Live View by selecting the **OFF** button.

---

# 6️⃣ Logout Procedure

To securely exit the system, follow these steps:

1. Click on the **Logout** option.  
2. Confirm the action by pressing **OK**.

---

# 7️⃣ Conclusion

By following this SOP, the IT (Infrastructure & Security) team can efficiently access and retrieve vehicle number plate information captured by ANPR cameras through the NVR system.

This method ensures continuity of ANPR monitoring and reporting even in the absence of an ANPR license in **HikCentral Professional**.
